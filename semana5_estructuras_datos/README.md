# Semana 5 - Estructuras de Datos

## Descripción

Este proyecto simula un sistema básico de atención de estudiantes utilizando estructuras de datos fundamentales en Python.

## Estructuras utilizadas

### Lista

Se utiliza para almacenar los nombres de los estudiantes registrados.

### Cola

Se utiliza para representar el orden de atención. El primer estudiante que llega es el primero en ser atendido.

### Pila

Se utiliza para guardar el historial de atención. El último estudiante atendido es el primero que aparece al revisar el historial.

### Diccionario

Se utiliza para almacenar información detallada de cada estudiante, como carrera, semestre y estado.

## Instrucciones para ejecutar

Abrir la terminal dentro de la carpeta del proyecto y ejecutar:

```bash
python main.py
```

## Evidencia esperada

El estudiante debe tomar captura de pantalla donde se observe:

- El archivo `main.py`.
- La ejecución del programa.
- El resultado en consola.

## Entrega

Subir el proyecto a GitHub y entregar el enlace del repositorio.

## Tarea de la Semana 5

### Consigna

Desarrollar un programa en Python que simule un sistema básico de atención de estudiantes, aplicando las estructuras de datos fundamentales: lista, cola, pila y diccionario.

### Instrucciones

1. Crear una lista con al menos 5 estudiantes.
2. Mostrar los estudiantes registrados.
3. Crear una cola de atención con esos estudiantes.
4. Atender a los estudiantes en el mismo orden en que llegaron.
5. Guardar cada estudiante atendido en una pila llamada `historial_atencion`.
6. Mostrar el historial de atención usando comportamiento de pila: el último atendido debe mostrarse primero.
7. Crear un diccionario con los datos de cada estudiante.
8. El diccionario debe incluir carrera, semestre y estado.
9. Consultar y mostrar la información de un estudiante.
10. Comentar el código para explicar dónde se usa lista, cola, pila y diccionario.
11. Ejecutar el programa.
12. Tomar captura del resultado.
13. Subir el proyecto a GitHub.

### Metodología

La actividad se realizará de forma práctica usando Python. El estudiante debe analizar qué estructura de datos se utiliza en cada parte del programa y explicar brevemente su función en el `README.md`.

### Criterios de evaluación

- Uso correcto de lista.
- Uso correcto de cola.
- Uso correcto de pila.
- Uso correcto de diccionario.
- Código ordenado y comentado.
- Resultado visible en consola.
- Proyecto subido correctamente a GitHub.
