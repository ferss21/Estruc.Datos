# Semana 5 - Proyecto Integrador de TAD

## Objetivo

La semana 5 integra lista, pila, cola y diccionario en un sistema de tickets académicos.

El objetivo es comprender cómo estas estructuras pueden trabajar juntas dentro de un programa pequeño y funcional. El sistema simula solicitudes académicas de estudiantes, su registro, su cola de atención, el historial de acciones y la búsqueda por código.

## Continuación de la Semana 4

En la semana 4 se estudiaron los TAD fundamentales de forma individual:

- Lista
- Pila
- Cola
- Correspondencia o diccionario

En la semana 5 no se repite la teoría básica. Ahora se aplican esas estructuras juntas en un sistema funcional para resolver un problema práctico.

## Estructura de la carpeta

- `ejemplos/`: contiene ejemplos pequeños y guiados de cada estructura.
- `actividad_clase/`: contiene el proyecto que se trabaja en clase.

## Orden recomendado de trabajo

1. Revisar y ejecutar los ejemplos.
2. Analizar qué estructura usa cada archivo.
3. Entrar a `actividad_clase`.
4. Ejecutar `main.py`.
5. Probar el menú.
6. Completar la evidencia en `README.md`.
7. Subir el trabajo a GitHub.

## Comandos de ejecución

```bash
cd semana_05_proyecto_integrador_tad

python ejemplos/01_diccionario_tickets.py
python ejemplos/02_cola_atencion.py
python ejemplos/03_pila_historial.py
python ejemplos/04_lista_resumen.py

cd actividad_clase
python main.py
```
