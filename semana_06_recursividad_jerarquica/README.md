# Semana 6 - Recursividad jerárquica

## Objetivo
Comprender cómo una función puede resolver un problema llamándose a sí misma con una versión más pequeña, aplicando esta idea a estructuras con niveles.

## Conceptos clave
- Recursividad
- Caso base
- Caso recursivo
- Profundidad
- Pila de llamadas
- Jerarquía
- Recorrido de carpetas

## Orden recomendado
1. Ejecutar los ejemplos.
2. Leer los comentarios de cada archivo.
3. Ejecutar la actividad de clase.
4. Tomar capturas.
5. Actualizar el README de la actividad.
6. Subir la carpeta a GitHub.

## Comandos de ejecución

```bash
cd semana_06_recursividad_jerarquica
python ejemplos/01_cuenta_regresiva.py
python ejemplos/02_factorial_visual.py
python ejemplos/03_suma_lista_recursiva.py
python ejemplos/04_recorrer_carpetas.py
cd actividad_clase
python main.py
```
