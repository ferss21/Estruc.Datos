# Actividad de Clase - Semana 6

## Consigna
Desarrollar y ejecutar un explorador jerárquico académico en Python utilizando recursividad. El programa debe recorrer una estructura de carpetas y archivos, mostrar niveles, contar archivos, buscar un archivo y calcular la profundidad máxima.

## Instrucciones
1. Abrir el repositorio en Visual Studio Code.
2. Entrar a la carpeta semana_06_recursividad_jerarquica.
3. Ejecutar primero los ejemplos de la carpeta ejemplos.
4. Entrar a actividad_clase.
5. Ejecutar python main.py.
6. Probar todas las opciones del menú.
7. Buscar un archivo existente.
8. Buscar un archivo inexistente.
9. Tomar capturas de pantalla.
10. Subir la carpeta a GitHub.

## Metodología
La actividad se realizará de forma práctica e individual. El estudiante primero ejecutará ejemplos pequeños para comprender caso base, caso recursivo y profundidad. Luego aplicará esos conceptos en un programa que recorre una jerarquía académica.

El estudiante debe explicar con sus palabras cómo la función baja por los niveles y cómo retorna los resultados.
