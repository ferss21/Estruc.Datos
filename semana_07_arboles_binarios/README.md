# Semana 7 - Arboles binarios

## Objetivo
Comprender la nomenclatura basica de los arboles binarios: raiz, nodo, padre, hijo, hoja, nivel, altura y recorrido. Tambien se implementa un arbol binario de busqueda con operaciones basicas.

## Conceptos clave
- Nodo
- Raiz
- Padre
- Hijo izquierdo
- Hijo derecho
- Hoja
- Nivel
- Altura
- Recorrido preorden
- Recorrido inorden
- Recorrido postorden
- Arbol binario de busqueda

## Orden recomendado
1. Verificar la version de Python.
2. Entrar a la carpeta de la semana.
3. Revisar la estructura de archivos.
4. Ejecutar cada ejemplo.
5. Ejecutar la actividad de clase.
6. Tomar capturas de pantalla.
7. Subir la carpeta a GitHub.

## Comandos para ejecutar en Fedora

```bash
python3 --version
cd semana_07_arboles_binarios
tree -L 2
python3 ejemplos/01_nodo_raiz.py
python3 ejemplos/02_arbol_manual.py
python3 ejemplos/03_recorridos.py
python3 ejemplos/04_busqueda_bst.py
python3 actividad_clase/main.py
```

## Descripcion de los ejemplos

### 01_nodo_raiz.py
Presenta la clase `Nodo`, crea una raiz y muestra su dato en consola.

### 02_arbol_manual.py
Crea un arbol binario pequeno de forma manual para identificar raiz, padre, hijos y hojas.

### 03_recorridos.py
Construye un arbol binario de busqueda y muestra los recorridos preorden, inorden y postorden.

### 04_busqueda_bst.py
Implementa insercion y busqueda en un arbol binario de busqueda.
