# Vocabulario de la Semana 9

| Término | Definición | Ejemplo |
|---|---|---|
| Conjunto | Colección de elementos únicos. | `{101, 102}` |
| Elemento | Cada valor almacenado en una colección. | `101` |
| Dato único | Valor que aparece una sola vez. | Un código de matrícula |
| Pertenencia | Relación que indica si un valor está en una colección. | `101 in inscritos` |
| Unión | Reunión de los elementos de dos conjuntos. | `a \| b` |
| Intersección | Elementos comunes entre dos conjuntos. | `a & b` |
| Diferencia | Elementos del primero que no están en el segundo. | `a - b` |
| Diferencia simétrica | Elementos que pertenecen a uno solo de los conjuntos. | `a ^ b` |
| Subconjunto | Conjunto cuyos elementos están dentro de otro. | `a.issubset(b)` |
| Superconjunto | Conjunto que contiene todos los elementos de otro. | `b.issuperset(a)` |
| Duplicado | Valor repetido dentro de los datos originales. | `[101, 101]` |
| set | Tipo de Python para representar conjuntos. | `set([101, 102])` |
| list | Colección ordenada que permite repetidos. | `[101, 101]` |
| dict | Colección de pares clave-valor. | `{101: 95}` |
| Hashable | Valor con identificador estable que puede pertenecer a un set. | `101`, `"ED101"` |
