# Tarea evaluada - Control de entregas por grupos

## Consigna

Complete el archivo `tarea_evaluada.py` para analizar la matrícula y las entregas de dos grupos. La solución debe utilizar conjuntos y presentar los resultados ordenados.

## Actividades

1. Eliminar los códigos duplicados de cada lista de matrícula.
2. Obtener los estudiantes inscritos en ambos grupos.
3. Encontrar los estudiantes que pertenecen solamente a uno de los grupos.
4. Calcular cuáles estudiantes todavía no entregaron.
5. Comprobar si todos los códigos con entrega pertenecen a la matrícula general.

## Condiciones

- Utilizar las operaciones de conjuntos estudiadas durante la semana.
- No cambiar los datos de prueba hasta comprobar el resultado esperado.
- Mantener la función `main()`.
- Agregar comentarios breves que expliquen las operaciones utilizadas.

## Evidencia

Entregue el archivo completado, una captura de su ejecución y una explicación breve de cada resultado.
