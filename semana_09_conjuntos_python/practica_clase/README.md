# Práctica de clase - Análisis de grupos académicos

## Consigna

Utilice el programa para administrar estudiantes inscritos, asistencia y entregas. Analice los resultados producidos por las operaciones entre conjuntos.

## Instrucciones

1. Ejecute `python practica_clase/main.py` desde la carpeta de la Semana 9.
2. Revise los datos iniciales mediante la opción 9.
3. Agregue y elimine códigos en cada registro.
4. Intente repetir un código y escribir un valor no numérico.
5. Compare la matrícula con un segundo grupo.
6. Compruebe los estudiantes pendientes y ausentes.

## Metodología

Trabaje primero con los datos iniciales. Antes de seleccionar una opción, escriba el resultado que espera. Después compare su predicción con la salida y explique qué operación se aplicó.

## Resultado esperado

El sistema debe mantener códigos únicos, rechazar entradas no numéricas, evitar registros académicos de estudiantes no inscritos y presentar resultados ordenados.

## Evidencia solicitada

- Captura del resumen inicial.
- Captura de un código duplicado rechazado.
- Captura de una entrada no numérica validada.
- Capturas de intersección, diferencia y comparación de grupos.
- Breve explicación de los resultados.

## Preguntas de reflexión

1. ¿Qué duplicados evita automáticamente el programa?
2. ¿Por qué asistencia y entregas deben ser subconjuntos de inscritos?
3. ¿Qué operación permite encontrar pendientes?
4. ¿Qué cambiaría si los datos se guardaran en listas?
